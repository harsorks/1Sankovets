## Описание задания
Разработка игры "Крестики-нолики" в двух версиях:
1. Версия 1.0 — Базовая версия игры с графическим интерфейсом (WPF).
2. Версия 2.0 — Расширенная версия со счетчиком побед и возможностью кастомизации символов игроков.

## Технологии
- Язык программирования: C#
- Платформа: WPF (.NET)
- Среда разработки: Visual Studio
- Система контроля версий: Git + GitHub Desktop
